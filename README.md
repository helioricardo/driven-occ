# driven-occ
 CLI to manage OCC development at Driven.cx

## How does it work?
### Install
```sh
npm i helioricardo/driven-occ
npx docc -s
```

### Using

#### Environment Commands

Config an enviroment
```sh
npx docc -e config
```

Change the enviroment
```sh
npx docc -e change
```

Get the current enviroment
```sh
npx docc -e current
```
